# 1.0 🐣
🔨 Xcode 14.3.1  
⌚️ watchOS 9.4  
☁️📦 SoundCloud Swift Package 0.0.1
  
### ✨ Features
- Sign in with SoundCloud account, refresh OAuth access token
- Play music
- Access playlists 
    - Liked tracks 
    - Recently posted 
    - Liked playlists 
    - My playlists
- Shuffle playlist
- Liked tracks pagination
- Like-unlike tracks
- Share tracks, playlists, my profile
- Change playback speed
- OS media controls
- **Minium watchOS: 9.1**
